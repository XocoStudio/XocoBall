# XocoBall (WebZSpheres)

Bienvenido a **XocoBall**, una potente herramienta de escultura y modelado orgánico basada en la técnica de "ZSpheres" (Esferas Z). Esta aplicación te permite crear personajes, criaturas y formas orgánicas complejas de manera rápida e intuitiva, comenzando con un simple esqueleto de esferas y terminando con una malla esculpida con alto nivel de detalle.

## ¿Qué puedes hacer con XocoBall?

### 1. Modelado Estructural (Modo Estructura)
El corazón de XocoBall. Crea el "esqueleto" de tu modelo usando esferas interconectadas.
*   **Creación Rápida:** Haz clic para añadir nuevas esferas y extruir ramas desde cualquier punto.
*   **Manipulación Total:** Mueve, Rota y Escala cada articulación (nodo) para definir la pose y proporciones perfectas.
*   **Simetría Avanzada:** Trabaja con simetría en los ejes X, Y, Z e incluso **Simetría Radial** para crear formas complejas como estrellas, flores o criaturas alienígenas con múltiples extremidades.

### 2. Generación de Piel (Skinning)
Convierte tu estructura de esferas en una malla 3D real con un solo clic.
*   **Piel Adaptativa (Fluida):** Genera una superficie suave y continua alrededor de tus esferas.
*   **Piel Voxel:** Crea una malla basada en vóxeles para un estilo más robusto o "bloqueado".
*   **Previsualización Instantánea:** Alterna rápidamente entre la vista de esferas y la vista de malla (Tecla `A`) para ver cómo quedará tu modelo.

### 3. Escultura Digital (Modo Esculpir)
Una vez generada la malla, pasa al modo de escultura para refinar los detalles.
*   **Pinceles Profesionales:**
    *   **Draw:** Añade volumen y detalles básicos.
    *   **Clay:** Acumula "arcilla" digital para dar forma muscular u orgánica.
    *   **Inflate:** Infla zonas específicas.
    *   **Smooth:** Suaviza superficies rugosas.
    *   **Flatten:** Aplana zonas para crear superficies duras.
    *   **Paint:** Pinta colores directamente sobre la superficie de tu modelo.
*   **DynaMesh:** Recalcula la geometría de tu modelo (Remesh) para redistribuir los polígonos y seguir esculpiendo sin deformaciones.

### 4. Materiales y Visualización
*   **Control de Material:** Ajusta la rugosidad (Roughness) y el aspecto metálico (Metalness) de tu creación.
*   **Wireframe:** Visualiza la malla de alambre para ver la topología de tu modelo.

### 5. Exportación y Guardado
*   **Guardar Escena:** Guarda tu progreso en formato JSON para continuar editando la estructura de esferas más tarde.
*   **Exportar Modelo:** Exporta tu creación final en formatos **.OBJ** o **.GLB** para usarla en motores de juegos, impresoras 3D u otro software de modelado (Blender, Unity, Unreal Engine).

## Atajos de Teclado Rápidos

*   **Q**: Herramienta Dibujar (Añadir esferas).
*   **W**: Herramienta Mover.
*   **E**: Herramienta Rotar.
*   **R**: Herramienta Escalar.
*   **X**: Activar/Desactivar Simetría.
*   **A**: Alternar Previsualización de Piel (Skin).
*   **Ctrl + Z / Ctrl + Y**: Deshacer / Rehacer.
